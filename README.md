## ✨ Overview

This project is a simple implementation of the classic game:

👉 Rock 🪨
👉 Paper 📄
👉 Scissors ✂️

It helps beginners understand core programming concepts in a practical way.

---

## 🎮 Game Preview

```
Enter your choice:
1. Rock
2. Paper
3. Scissors
Your choice: 2
Computer chose: 1
Result: 🎉 You Win!
```

---

## 📌 Rules

| Your Choice | Computer | Result  |
| ----------- | -------- | ------- |
| Rock        | Scissors | ✅ Win   |
| Scissors    | Paper    | ✅ Win   |
| Paper       | Rock     | ✅ Win   |
| Same        | Same     | 🤝 Draw |

---

## 🚀 Run This Project

### 🔧 Compile

```bash
g++ main.cpp -o game
```

### ▶️ Execute

```bash
./game
```

## 🧠 What You Will Learn

✔ Object-Oriented Programming (OOP)
✔ Class & Object (`class Game`)
✔ Conditional Logic (`if-else`)
✔ Random Number Generation (`rand()`)
✔ Input Validation
